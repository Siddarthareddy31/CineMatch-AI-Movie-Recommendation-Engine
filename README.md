#  CineMatch AI - Movie Recommendation Engine

A content-based movie recommendation system that suggests similar movies using Machine Learning and Natural Language Processing techniques.

##  Live Demo
https://cinematch-ai-movie-recommendation-engine-bmmmhydqofdqfbz66xn6s.streamlit.app/

##  Key Features

- Recommends top 5 similar movies based on content similarity.
- Uses TMDB 5000 Movies Dataset containing 5000+ movies.
- Performs feature engineering on genres, cast, crew, keywords, and movie overview.
- Uses CountVectorizer for text vectorization.
- Computes Cosine Similarity to identify related movies.
- Fetches movie posters and details using TMDB API.
- Interactive web application built with Streamlit.
- Deployed online for real-time usage.

##  Machine Learning Pipeline

1. Data Collection and Preprocessing
2. Feature Engineering
3. Text Vectorization using CountVectorizer
4. Similarity Matrix Generation
5. Recommendation System Development
6. Streamlit Deployment
